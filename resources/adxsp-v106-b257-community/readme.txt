Ribbon Designer for SharePoint and Office 365
-------------------------------------------------------------------------------

SHORT DESCRIPTION
-------------------------------------------------------------------------------
Ribbon Designer for SharePoint and Office 365 is a .NET component set that
simplifies customizing and extending the Ribbon UI of SharePoint and
Office 365.


INSTALLATION PROCEDURE
-------------------------------------------------------------------------------
1. If you are upgrading from the previous version of Ribbon Designer, please,
   uninstall your current version first.
2. Close all Visual Studio instances.
3. Run the included setup.exe file and install the pack contents to your hard
   drive by following the instructions of the installation wizard. Ribbon
   Designer setup installs all packages automatically.
